How to Play

First, select a game mode. There are two game modes: Infinite & Normal. In the Infinite game mode, 20 songs are played. In the Normal game mode, 3 songs are played. Then, select a genre. In both game modes, Each song will play for 5 seconds and after the song ends you have ten seconds to guess. You may request for the song to continue playing. To exit the game, say "exit". When you want to start playing, tell Alan!

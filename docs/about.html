About

Hi there, my name is Johnny. I made this online game and integrated it with Alan AI 's voice assistant to elevate the standard "Guess the Song" game. So instead of you having to type in the name of the artist or the song, all you have to do is speak. This seemingly small change actually allows the flow of the game to feel more natural and the game itself to become more accessible. If you would like to learn more, this is my project link: Project. And if you want the code, this is my Github link: Github.
